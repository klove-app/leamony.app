RunTracker Bot User Guide v2.0

Getting Started

To start using RunTracker Bot:

  1. Go to @sl_run_bot
  2. Send the /start command to activate the bot
  3. Add the bot to your running chat (if you're an admin)

Basic Commands

/start - Getting Started
Activates the bot and shows a welcome message with basic commands.
Recording a Run
To record a run, mention the bot (@sl_run_bot) and specify the distance in your group chat.
Example: @sl_run_bot 5.2
